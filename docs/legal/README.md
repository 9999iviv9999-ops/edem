# Юридические документы Vprok (черновики)

**Важно.** Тексты ниже — **не юридическая консультация**. Это структурные черновики под модель «платформа отложенной выдачи товаров» (предоплата, комиссия платформы, ритейлер как продавец). Их необходимо **согласовать с юристом** в РФ (или иной релевантной юрисдикции), подставить реквизиты оператора, уточнить налоговую и платёжную модель, признание вас маркетплейсом/агентом/иным статусом.

## Плейсхолдеры (заменить перед публикацией)

- `[НАЗВАНИЕ_ОПЕРАТОРА]` — полное наименование ИП или ООО
- `[ИНН]`, `[ОГРН_ИЛИ_ОГРНИП]`
- `[ЮРИДИЧЕСКИЙ_АДРЕС]`
- `[EMAIL_ПОДДЕРЖКИ]`, `[САЙТ_ПЛАТФОРМЫ]` (например `https://vprok.club`)
- `[ДАТА_ВСТУПЛЕНИЯ]` — дата публикации редакции

## Связь с продуктом

- Комиссия платформы по умолчанию **3%** (`platformFeeBps = 300`) — в документах указано как ориентир; фактическое значение должен совпадать с UI/API и договором с ритейлером.
- Ритейлер публикует свои условия (`termsUrl`, `returnPolicyText` и т.д.) — в офертах зафиксировано, что **условия продажи** определяются продавцом, платформа — инфраструктура.
