# Skinulis Web skeleton

Mobile-first frontend app for:
- campaign feed
- campaign card
- create campaign
- placement payment
- donate flow
- complaint submission
- campaign report upload

## Suggested pages

- `/` - feed
- `/campaign/:id` - campaign details
- `/create` - create campaign
- `/checkout/publish/:id` - pay placement fee
- `/checkout/donate/:id` - donate
- `/profile` - owner profile
- `/profile/campaigns/:id/report` - upload report

## UI rules

- show quick amount chips: 10 / 30 / 50 / 100 RUB
- sticky CTA on campaign page
- visible report and complaints section on each campaign
