# STARTFLOW Forum

Готовый изолированный форум стартапов с обязательными разделами:

1. Стартап в виде идеи
2. Релиз стартапов
3. Краудфандинг
4. Продажа стартапов

Форум построен как система независимых разделов: пользователи публикуют темы
сразу в нужной категории по своей задаче (без обязательного перехода между разделами).

## Правила и монетизация

- Для каждого раздела есть отдельные правила публикации на главной странице.
- Платформа не участвует в сборе средств и не принимает платежи пользователей.
- Все финансовые операции идут напрямую между участниками.
- Тарифы доступа (криптовалюта):
  - `Обычный пользователь` — `$10`
  - `Gold` — `$50`
  - `Premium` — `$70`

## Быстрый запуск

```bash
cd startup-forum
npm start
```

Открыть в браузере:

- `http://127.0.0.1:4211`

## Что внутри

- `server.js` - backend API + static server (без внешних зависимостей)
- `public/` - интерфейс форума
- `data/forum-db.json` - локальная JSON-база
- `DOMAIN_GUARD.md` - правила изоляции доменов/нейминга

## API

- `GET /api/meta` - метаданные форума и sections
- `GET /api/posts` - список постов
- `POST /api/posts` - создать пост
- `PATCH /api/posts/:id` - обновить раздел/статус

## Гарантия непересечения с другими проектами

- уникальный namespace: `startflow-forum`
- отдельный локальный порт: `4211`
- не используются домены и имена текущих продуктов репозитория
